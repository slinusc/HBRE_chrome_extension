# Privacy Policy for H&B Property Assistant

**Last Updated**: December 4, 2024

## Extension Purpose
H&B Property Assistant is an internal productivity tool for H&B Real Estate professionals to manage property listings for tenant representation.

## Data We Collect
- User login credentials (email/password) for H&B backend authentication
- Selected project preferences
- Property listing information from ImmoScout24 and Homegate (address, price, description, images)

## How We Use Data
- **Authentication data**: To verify user identity with H&B backend system
- **Property data**: To save listings to user's selected H&B projects
- All data is sent directly to the user's configured H&B backend server
- This extension reduces manual data entry by automating the transfer of publicly available property information

## Data Storage
- Login tokens are stored locally in browser storage only
- No data is stored on third-party servers
- All property data is transmitted only to the configured H&B backend URL

## Data Sharing
- We do not share data with third parties
- Data is only transmitted between the extension and the user's H&B backend system
- No analytics, tracking, or third-party services are used

## User Control
- Users can log out at any time to clear stored authentication tokens
- Users control which H&B backend URL the extension connects to
